Router Demo App

Instructions

  1. Set up Routing in the app.routing.module.ts
  2. Configure the router outlet and links in the app.component.html
  3. Configure the list items in the products.component.html to link to the details component with each ID as a parameter
  4. Configure the detail.component.ts to dynamically show the correct product based on the route parameter

Refer back to the presentation, whenever you are unsure. There should be examples for everything asked for you here. If you want even more information about routers and what exactly you are doing, the angular documentation provides a lot of detail.
